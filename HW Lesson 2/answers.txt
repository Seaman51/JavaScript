Задача ?1. Почему код даёт именно такие результаты?

var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

Ответ.
1. В первой строке оператором var мы обьявляются переменные a, b, с и d. В a и b заносятся значения 1.
2. Во второй строке префиксным унарным оператором ++a происходит инкрементирование (прибавление единицы) к
значению переменной а (числу 1) и запоминание результата (число 2) в переменнные с и a, затем командой alert
производится вывод полученного значения переменной с на экран.
3. В третьей строке постфиксным унарным оператором b++ происходит присвоение значения b переменной d (1) и 
затем инкрементирование (прибавление единицы) к значению переменной b (числу 1) и вывод значения переменной d
(число 1) командой alert на экран. В переменную b запоминается результат инкрементирования (цифра 2).
4. В четвёртой строке сначала происходит инкрементирование переменной а, которая становится = 3, затем к ней
прибавляется 2 и сумма (число 5) заносится в переменную c. Значение переменной с выводится на экран командой
alert.
5. В пятой строке кода происходит сложение числа 2 и переменной b (цифра 2). Сумма (4) запоминается в ячейку d
и выводится на экран. А в переменную b запоминается результат инкрементирования (цифра 3).
6. Шестой и седьмой строкой командами alert на экран поочерёдно выдаются значения переменных a и b (цифры 3).


Задача 2. Чему будет равен x в примере ниже?

var a = 2;
var x = 1 + (a *= 2);

Ответ: 5.
Вторую строчку можно расшифровать как х = 1 + (а = а * 2), то есть помимо того, что вычисляется и запоминается
значение х, также вычисляется и запоминается новое значение переменной а оператором присваивания с умножнением.


Задача 3. Объявить две целочисленные переменные a и b и задать им произвольные начальные значения. Затем написать
скрипт, который работает по следующему принципу:
* если a и b положительные, вывести их разность;
* если а и b отрицательные, вывести их произведение;
* если а и b разных знаков, вывести их сумму;
ноль можно считать положительным числом.

Ответ:
let a = 0, b = 2;
if (a>=0 && b>=0) {
    alert (a-b);
}else if (a<0 && b<0) {
    alert (a*b);
}else if (a<0 && b>=0 || a>=0 && b<0) {
    alert(a+b);
}

Задача 6. Реализовать функцию с тремя параметрами: function mathOperation(arg1, arg2, operation),
где arg1, arg2 - значения аргументов, operation - строка с названием операции. В зависимости от
переданного значения операции выполнить одну из арифметических операций (использовать функции из
пункта 3) и вернуть полученное значение (использовать switch).

Ответ:
function mathOperation(arg1, arg2, operation) {
    switch(operation) {
        case "+":
            alert (arg1+arg2);
            break;
        case "-":
            alert (arg1-arg2);
            break;
        case "*":
            alert (arg1*arg2);
            break;
        case "/":
            alert (arg1/arg2);
            break;
        default:
            alert("Некорректно введенная опция!");
    }
}
mathOperation(2,3,"*")

Задача 8*. С помощью рекурсии организовать функцию возведения числа в степень.
Формат: function power(val, pow), где val - заданное число, pow - степень.. 

Ответ:
function power(val, pow) {
    if (pow != 1) {
        return val * power (val, pow-1);
    }else{
        return val;
    }
}
alert("2 в 4 степени = " + power(2,4));
